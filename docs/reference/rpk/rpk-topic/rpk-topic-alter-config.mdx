Set, delete, add, and remove key/value configurations for a topic. This command lets you incrementally alter the configuration for multiple topics at the same time. Incremental altering supports the following operations:

- Set a key=value pair
- Delete a key's value
- Append a new value to a list-of-values key
- Remove an existing value from a list-of-values key

### Usage

```
rpk topic alter-config [TOPICS...] --set key=value --delete key2,key3 [flags]
```

For example, to change the replication factor on an existing topic: 

```
rpk topic alter-config [TOPICS...] --set replication.factor
```

The updated replication factor also applies to new partitions created in the topic.
